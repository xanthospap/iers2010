1. Purpose of this document

This document describes the libiers10++ Software Package and serves as a reference manual. There should be an application programming interface (API) shipped with the Package, located at iers2010/doc/html; you can open the file index.html with your favorite web browser and serach through the API. In case this folder is empty (i.e. iers2010/doc/html), you can create the API via running doxygen using the configuration filedoxy.conf. In any Unix-type system,just open a terminal/konsole and execute the following: 

bash$ doxygen doxy.conf

If doxygen is not available in your system, you can download it from this link. Note that using doxygen and modifing the config file, you can create the documentation in various formats (e.g. ps, latex, html, docbook, etc).
